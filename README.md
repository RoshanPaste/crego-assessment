# Crego Assessment Test

## Installation

1. Clone the repository to your local machine using:

   ```bash
   git clone https://github.com/RoshanPaste/crego-assessment.git
2. Change to the project directory:

   ```bash
   cd crego-assessment

3. Install the required dependencies:

    ```bash
    npm install
4. Run the project

   ```bash
   npm run dev
